##  Overview
This repository serves as a testing ground for **quantitative trading strategies** and **financial automation tools**. It documents my ongoing research into market microstructure, technical analysis, and algorithmic execution using Python.

The goal of this project is to engineer robust trading logic, optimize parameters via backtesting, and analyze risk-adjusted returns.
